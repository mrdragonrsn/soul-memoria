# Plugins e Repositórios Úteis

## Obsidian AI Plugins

### 1. AI Plugin (Oficial)
- **ID**: Maximophone/obsidian-ai-plugin
- **O que faz**: Integração com múltiplos provedores de IA
- **Provedores**: OpenAI, Claude, Gemini, DeepSeek, Perplexity
- **Models**: 50+ modelos
- **Instalação**: Community Plugins →Buscar "AI"

### 2. Steward
- **ID**: googlicius/obsidian-steward
- **O que faz**: Agente autônomo com comandos AI
- **Provedores**: OpenAI, Gemini, DeepSeek, Ollama
- **Funcionalidades**: Pesquisar, criar notas, automação

### 3. SystemSculpt
- **Site**: systemsculpt.com
- **O que faz**: AI automations dentro do vault
- **Funcionalidades**: PDF → Markdown, transcrever áudio

### 4. Claude Code Plugin
- **Repositório**: deivid11/obsidian-claude-code-plugin
- **O que faz**: Claude Code direto no Obsidian
- **Backend**: Claude Code ou OpenCode

### 5. Claudian
- **Repositório**: YishenTu/claudian
- **O que faz**: Agente Claude no vault
- **MCP**: Suporte a servers externos

### 6. ACP Client
- **Repositório**: WhiskeyJack96/obsidian-acp
- **O que faz**: Agent Client Protocol
- **Agentes**: Claude Code, Gemini CLI, OpenCode

## Links de Instalação

### Obsidian Plugins
1. Abrir Obsidian → Settings → Community Plugins
2. Buscar pelo nome
3. Install e Enable

### GitHub (Manual)
1. Baixar latest release
2. Extrair para .obsidian/plugins/
3. Reload e enable
