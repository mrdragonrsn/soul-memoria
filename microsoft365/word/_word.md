# Word

Editor de textos e documentos.

## Para que serve
Criar textos, documentos, currículos, relatórios e artigos.

## Como usar
Para criar documentos profissional:
- Estruturar com títulos claros
- Usar sumário automático
- Inserir imagens quando necessário

## Exemplos
- Currículo (CV)
- Relatórios
- Artigos
- Contratos
