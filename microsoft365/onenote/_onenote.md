# OneNote

Anotações digitais.

## Para que serve
Criar anotações, Notas e organização de ideias.

## Como usar
Para usar OneNote Effectively:
- Criar cadernos por tema
- Usar tags para organizar
- Sincronizar entre dispositivos

## Exemplos
- Notas de reunião
- Ideias de projetos
- Rascunhos
