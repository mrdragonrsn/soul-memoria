# PowerPoint

Apresentações visuais.

## Para que serve
Criar apresentações, slides e demonstrações visuais.

## Como usar
Para criar apresentações eficazes:
- Usar design consistente
- Texto direto e objetivo
- Imagens e gráficos

## Exemplos
- Apresentação de projetos
- Slides explicativos
- Demonstrações
