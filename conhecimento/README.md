# Base de Conhecimento

Este diretório contém conhecimento compartilhado entre as IAs.

## Estrutura Sugerida
- `projetos/` - Conhecimento específico de projetos
- `padroes/` - Padrões e convenções do codebase
- `ferramentas/` - Informações sobre ferramentas usadas

## Como Adicionar
Para adicionar conhecimento:
1. Criar arquivo `.md` nesta pasta
2. Nomear com nome descritivo
3. Atualizar este README com referência
