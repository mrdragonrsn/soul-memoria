# O que foi criado na Memória E:\IA

## Visão Geral
Este vault (E:\IA) é a memória compartilhada entre as IAs. Foi organizado para funcionar como um segundo cérebro com sincronização automática via Git.

## Estrutura de Pastas

- `long_prazo.json` - Dados estruturados JSON

### Contexto (contexto/)

## Regras Importantes

### 1. Extensão de Arquivos
**NUNCA** usar `.txt`. Sempre usar `.md` (Markdown).

### 2. Sincronização Git
Sempre que atualizar qualquer arquivo, sincronizar com Git:
```powershell
Push-Soul "update: descrição"
```

### 3. Nome dos Arquivos
Evitar `index.md`. Usar nomes relevantes como `_nome.md`.
